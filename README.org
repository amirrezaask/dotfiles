** Rebuild my system
Since I have all my configurations as org files, I can do a refresh with just this simple code block
#+begin_src emacs-lisp
  (dolist (file '(
  	       "Emacs.org"
  	       "ZSH.org"
  	       "Tmux.org"
  	       "Ghostty.org"
  	       "Git.org"
  	       ))
    (org-babel-tangle-file file))
#+end_src
