# My neovim IDE
## Features 
### Navigation 
- Goto definition ( gd )
- Find References ( gr )
- Find Implementation ( gi )
### Editing
- Autocomplete
- Snippets
- Rename ( Shift+R )
### Actions
- Build ( ,ab )
- Test ( ,ata )
- Run ( ,ar )
- Debugging
- Format ( ,af )
- or any other action

### Git
- See changed files ( Ctrl + K )
- commit changes ( chunk or file ) ( Ctrl + k )
