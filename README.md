# Dotfiles
My personal computing experience.

## Installation
```bash
git clone https://github.com/amirreza/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./sync
```

### Supported Platforms
- [Ubuntu](https://ubuntu.com/)
- [Fedora](https://fedoraproject.org/)
- [macOS](https://www.apple.com/os/macos/)
